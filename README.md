**RC Phase Shift Oscillator**

This project demonstrates the working of an RC Phase Shift Oscillator using a transistor amplifier stage and a feedback network consisting of three RC sections.

Theory:
An RC phase shift oscillator produces sinusoidal output by providing 180° phase shift through RC network and 180° phase shift from the amplifier.

Tools Used:
- KiCad
- LTspice (for simulation)

Output:
Stable sinusoidal waveform generated.# rc-phase-shift-oscillator
